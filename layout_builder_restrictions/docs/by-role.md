# Restrictions by role

To restrict access to layouts or blocks based on user role, sites can use [Layout Builder Restrictions by Role](https://www.drupal.org/project/layout_builder_restrictions_by_role), which provides a Layout Builder Restrictions plugin.

Developers can also implement hooks to restrict by role, as described in [Custom restrictions using hooks](https://project.pages.drupalcode.org/layout_builder_restrictions/hooks/)
